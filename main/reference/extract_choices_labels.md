# Extract labels from choices basing on attributes and names

Extract labels from choices basing on attributes and names

## Usage

``` r
extract_choices_labels(choices, values = NULL)
```

## Arguments

- choices:

  (`list` or `vector`)  
  select choices

- values:

  optional  
  choices subset for which labels should be extracted, `NULL` for all
  choices.

## Value

(`character`) vector with labels
